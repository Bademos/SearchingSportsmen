Типы сообщений:
1. Регистрация (UM <--> SM)
	1.1 Успех
		{	
			message: "Пользователь с email:{email} успешно зарегистрирован"
		}
	1.2 Неудача 

		{	
			message: "Возникли проблемы при регистрации"
		}
2. Вход (UM <--> SM)
	2.1 Успех
		{	
			message: "Пользователь с email:{email} успешно вошел в аккаунт:{user_id}"
		}
	2.2 Неудача 

		{	
			message: "Возникли проблемы при аутентификации
		}

3. Вход (UM <--> SM)
	3.1 Успех
		{	
			message: "Пользователь с email:{email} успешно сменил пароль"
		}
	3.2 Неудача 

		3.2.1 	{	
				message: "Возникли проблемы при смене пароля"
			}
		3.2.2 	{	
				message: "Пользователя с email:{email}  нет в базе данных"
			}

	3.3 Непереход по ссылке
		3.2.1 	{	
				message: "Время действия ссылки для смены пароля окончено"
			}
